# Titile

> Description

## Installation

> Install dependencies

``` bash
yarn add
```

## Complete setup

> Build for development with debug

``` bash
yarn run start
```

> Build for production with minification

``` bash
yarn run build
```

## Tasks setup

### EJS

> Build task for development with debug

``` bash
yarn run ejs:dev
```

> Build task for production with minification

``` bash
yarn run ejs:prod
```
