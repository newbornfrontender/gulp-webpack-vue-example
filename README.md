# Titile

> Description

## Installation

Install dependencies

``` bash
yarn add
```

## Build complete setup

Build for development with serve, debug, etc

``` bash
yarn run start
```

Build for production with minification

``` bash
yarn run build
```

## Tasks setup

### EJS

> Description

Build "ejs" task for development

``` bash
yarn run ejs:dev
```

Build "ejs" task for production

``` bash
yarn run ejs:prod
```
